# 🚀 Day 09 - Nginx Reverse Proxy with Flask & MySQL

## 📖 Project Overview

This project demonstrates a production-style deployment of a Flask Employee Management System using Docker, MySQL, and Nginx Reverse Proxy.

## 🛠 Tech Stack

- Docker
- Docker Compose
- Flask
- MySQL
- Nginx
- HTML
- CSS
- Jinja2
- Google Cloud VM

## ✨ Features

- Add Employee
- View Employees
- Edit Employee
- Delete Employee
- Premium Dashboard UI
- Nginx Reverse Proxy
- Persistent MySQL Database
- Responsive Layout

## 🚀 Run the Project

```bash
docker compose up --build -d
```

Open in browser:

```
http://YOUR_VM_EXTERNAL_IP
```
